***********************
Heaviside step function
***********************

The `Heaviside step function`_ can be defined as

.. math::
    :nowrap:

    \begin{equation}
        H(x) = \begin{cases} 0, &x < 0, \\ 1, &x \gt 0. \end{cases}
    \end{equation}
